CONTACT PAGE  (app/contact/page.tsx + components/ContactForm.tsx)
================================================================================

CONTACT FORM — fields & controls
  input     Name              (text, required, maxLength 80,  autoComplete=name)
  input     Email             (email, required, maxLength 254, autoComplete=email)
  input     Company           (text, optional, maxLength 120, autoComplete=organization)
  select    Budget            (options below, optional)
              - Under $25k
              - $25k – $50k
              - $50k – $150k
              - $150k+
              - Not sure yet
  input     Message           (textarea, required, min 20 / max 4000 chars)
  checkbox  "I agree to PhiBrain processing my message…" (required)
  link      "privacy policy"  (anchor)              → #privacy
  submit    "Send message"    (primary button)      → POST /api/contact
  button    "Send another message" (on success)     → resets form to idle

  Hidden anti-spam controls (not user-clickable):
  hidden    _hp (honeypot)
  hidden    csrf (mirrors csrf-token cookie)
